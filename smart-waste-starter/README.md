## Setup Backend
cd backend
cp .env.example .env    # edit env if needed
npm install
mkdir uploads           # make upload dir for dev
npm run dev

## Setup Frontend
cd frontend
npm install
npm run dev

Open frontend at the address shown by Vite (default http://localhost:5173) and backend at http://localhost:5000

Register a user, login, then go to Report to upload a report. Admin dashboard shows reports on map.
